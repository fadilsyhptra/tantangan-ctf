# MISTERI HILANGNYA RESEP LEGENDA: WARUNG NASI PAK DENGKLEK

LAPORAN INSIDEN: MISTERI HILANGNYA RESEP LEGENDA
WARUNG NASI PAK DENGKLEK
                
Buset dah, asli kacau balau! Pagi ini seisi Warung Nasi Pak Dengklek 
geger total. Gimana enggak? Pas Pak Dengklek mau ngeracik bumbu subuh-
subuh tadi, brankas digital tempat nyimpen kertas "Resep Rahasia 
Sambel Terasi" udah dalam keadaan terkunci rapat dan sistemnya 
diubah total! Resep fisik di dalamnya udah amblas gak bersisa. 

Apes bener, padahal siang ini ada kunjungan dari kritikus makanan 
internasional. Kalau sambel terasi itu gak ada, bisnis Pak Dengklek 
bisa terancam gulung tikar!

Kecurigaan langsung mengarah ke satu nama: "Udin". 
Dia itu mantan karyawan bagian dapur yang baru aja dipecat kemaren sore 
karena ketahuan sering ngemilin kerupuk kaleng tanpa izin. Pas dipecat, 
si Udin sempet ngancem sambil teriak, "Liat aja lu Pak, bakal gua bikin 
hambar ini warung!". Ditambah lagi, anak-anak dapur tau kalau si Udin 
ini dulunya anak tongkrongan warnet yang lumayan ngerti soal coding 
dan komputer.

Tim IT warung udah nyoba meriksa komputer utama di kantor Pak Dengklek. 
Sialnya, si Udin pinter. Dia udah nge-wipe hampir semua log 
sistem. Tapi, dia sengaja ninggalin satu folder mencurigakan di 
desktop yang berisi beberapa berkas aneh. 

Di dalam folder itu, tim IT cuma nemuin:
1. Sebuah skrip bernama 'brankas.py'.
2. Sebuah file dokumen yang cuma bisa dibaca sebagai teks biasa, 
   yaitu berkas investigasi berisi data pribadi si Udin sendiri.
3. Sebuah file audio aneh

Ancaman Lewat SMS
Tepat jam 7 pagi, Pak Dengklek menerima SMS dari nomor tidak dikenal:
"Mau resepnya balik, Pak? Coba temukan teks unik (Flag) di suatu file 
sebelum jam makan siang dimulai. Jangan coba-coba ulur waktu, atau 
resep itu bakal saya hapus permanen dari dunia ini dan warung lu bakal 
hambar selamanya. Detik terus berjalan, jangan sampai terlambat. HAHAHA!"

Pihak berwajib udah nyerah karena gak paham coding cok (cupu HAHAHA).
Sekarang,nasib Sambel Terasi Pak Dengklek ada di tangan lu! Coba lu analisis 
semua file yang ditinggalin si Udin. Selidikin identitasnya, temukan celah 
buat ngebongkar brankas digital itu, dan selametin resepnya sebelum 
jam makan siang dimulai! 

Gua ngandelin lu. Buruan sikat!
